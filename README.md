1. What is JSX, and why is it used?
ANSWER : JSX is a syntax extension for JavaScript that allows writing HTML-like code inside React. It makes UI code easier to read and describe component structure.

2. What is the difference between State and Props?
ANSWER : State is internal data managed by a component and can change over time, while Props are read-only data passed from a parent component to a child.

3. What is the useState hook, and how does it work?
ANSWER : useState is a React hook used to create and update state in functional components; it returns a state value and a function to update it.

4. How can you share state between components in React?
ANSWER : State is shared by lifting it up to a common parent component and passing it to children via props.

5. How is event handling done in React?
ANSWER : Event handling is done using camelCase event attributes and passing a function as the event handler.
